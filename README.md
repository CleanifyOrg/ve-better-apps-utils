## Prerequisites

- Node.js installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).

## Setup

```bash
yarn install
```

## Usage

Navigate to the project directory:

    ```bash
    cd sybil-dan
    ```

Distribute / Withdraw tokens:

    ```bash
    yarn distribute
    ```

Vote for random apps:

    ````bash
    yarn cast-round-vote
    ````
